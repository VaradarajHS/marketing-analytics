# 📊 Marketing Campaign Analysis Dashboard

## 📌 Project Overview

This project analyzes a bank's direct marketing campaign dataset to uncover customer demographics, campaign performance, and marketing trends. The analysis was performed using Python for data cleaning, SQL for querying, and Tableau for interactive dashboard development.

The goal of this project is to help marketing teams understand customer behavior, optimize campaign strategies, and identify trends that can improve future marketing efforts.

---

## 🎯 Objectives

- Analyze customer demographics.
- Understand campaign performance across different months.
- Identify customer contact preferences.
- Analyze customer financial characteristics.
- Build interactive dashboards for business decision-making.

---

## 🛠 Tools & Technologies

- Python (Pandas)
- SQL (MySQL)
- Tableau

## 📌 Key Insights

- Campaign activity peaked during specific months.
- Cellular communication was the most commonly used contact method.
- Married customers represented the largest customer segment.
- Customers with higher education generally maintained higher account balances.
- Average customer age was approximately 41 years.
- Average call duration was around 257 seconds.
- Customers were contacted approximately 2.7 times on average.
